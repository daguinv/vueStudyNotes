# 创建Vue实例传入的options
-  目前掌握这些选项
    - el:
        - 类型：string | HTMLElement -->可以是css选择器的字符串也可以是一个dom节点元素
        - 作用： 决定vue实例管理那一个dom 
    - data:
        - 类型：Object | Function（组件当中的data必须是一个函数）
        - 作用：Vue实例对应的数据对象
    - methods：
        - 类型： function
        - 作用： 定义属于Vue的一些方法，可以在其他地方调用，也可在指令中调用
